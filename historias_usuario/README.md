# Sistema de Gestión de Inventario Inteligente para Emprendedores Locales

## Descripción Breve
Una plataforma intuitiva diseñada para que los emprendedores locales gestionen el stock de sus productos en tiempo real, automaticen alertas de desabastecimiento, registren ventas diarias y visualicen reportes dinámicos que faciliten la toma de decisiones estratégicas.

## Lista de Funcionalidades (Flujo Guiado)
* **Registro de Emprendimiento:** Crear una cuenta de negocio ingresando el nombre del emprendimiento, correo electrónico, RUC/cédula y contraseña segura.
* **Autenticación y Acceso:** Inicio de sesión seguro mediante validación de credenciales con bloqueo automático tras 3 intentos fallidos.
* **Control de Stock y Catálogo:** Registrar, editar y eliminar productos especificando nombre, categoría, cantidad actual, costo unitario y precio de venta.
* **Alertas Inteligentes de Reabastecimiento:** Notificaciones automáticas visuales cuando un producto alcance el stock mínimo configurado.
* **Registro de Ventas Rápidas:** Simular un punto de venta seleccionando productos del inventario, aplicando cantidades, calculando el total y actualizando el stock automáticamente.
* **Reportes de Rendimiento:** Consultar gráficos dinámicos con el balance de ingresos, productos más vendidos y el valor total estimado del inventario actual.
